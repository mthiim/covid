COVID-19 status in Denmark by regions

This page shows time series data of COVID-19 infections country-wise and in the different Danish regions. The data has been smoothed using 7-day moving averages.

Some graphs show the daily number of infected whereas others shows it normalized against population size (same measure as Ministry of Foreign Affairs is using).

The raw data is fetched directly from Statens Serum Institut (SSI) and will update when they publish a new file (usually happens on weekdays around 14).

Note: Since the reported infections for a given date are often incomplete for 1-2 days, we ignore 2 days by default.
